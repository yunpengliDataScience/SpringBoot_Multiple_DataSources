This application demonstrates accessing 2 entities (User & Essay) from 2 different data sources by Spring Boot.
The databases, EssayH2 and UserH2, are H2 databases.

-----------------------------------------------------------------------
Access application:
http://localhost:7777/SpringBoot_Multiple_DataSources/

-----------------------------------------------------------------------
Spring Multiple Data Sources:

see application.properties, Database1Config.java, and Database2Config.java
